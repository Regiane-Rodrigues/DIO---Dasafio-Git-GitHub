
 ***Introdução .NET***
 
 
Plataforma de desenvolvimento open source criado pela Microsoft para desenvolvimento unificado de aplicações.Multiplataforma.

Linguagens:
c#
F#
virtual basic

Desenvolvimento da plataforma começou em 1998 


 ***Diferança entre .NETframework(legado) e .NET***

 .NET framework está atrelado apenas na plataforma Windows.
	
 .NET é mutliplataforma. 


 ***Funcionamento do compilador***

Compilador: é um progrma que realiza a conversão de linguagem de alto nível para de baixo nível.

Transpilador: é a conversão de uma linguagem ou implementação para outra. A sua saída permanece em linguagem de alto nível. Ex: typescrip para javascript.

Lingugem compilado: código fontee traduzido para linguagem de máquina.
Linguagem interpretada: São linguagens que fazem a leitura e interpretação diretamente do código fonte. Ex: javascript, php.
